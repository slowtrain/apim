개요
기관별 API 호출 가능 정책을 제한함
(단, 기관에 속한 모든 앱은 정책을 공유함)
Ex) 기관이 앱1, 앱2, 앱3 을 사용하고 있고,
10,000건/일의 사용량을 할당받은 정책을 사용하면
앱1, 앱2, 앱3 에서 호출한 API 건수는 합산하여 10,000/일 건을 넘지 못하게 제어함.
설정항목
1. 제목(필수) : 정책명
2. 정책 설명 : 정책에 대한 내용
3. 할당량 설정 : 초, 분, 시, 일, 월별 할당량을 제한함.
(기관에 등록된 여러 앱에서 API 호출 건수 모두 합산한 건수로 제한함)
