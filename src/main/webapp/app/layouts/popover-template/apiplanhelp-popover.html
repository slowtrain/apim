개요
기관에 속한 앱별+사용가능한 API별로 API 호출 가능 정책을 제한함
(단, API사용정책은 기관 사용정책 제한과 별개로 작동한다.)
Ex) 앱1,앱2,앱3에 각각에 API#1 호출 가능 정책을 부여함.
앱1은 API#1에 대해 10,000건/일의 사용양을 가짐.
앱2은 API#1에 대해 10,000건/일의 사용양을 가짐.
앱3은 API#1에 대해 10,000건/일의 사용양을 가짐.
각 앱별 API#1을 호출할 수 있는 사용량을 개별적으로 부여받아
앱별로 API#1 호출을 10,000건/일 을 넘지 못하게 제어함.
설정항목
1. 제목(필수) : 정책명
2. 정책 설명 : 정책에 대한 내용
3. 할당량 설정 : 초, 분, 시, 일, 월별 할당량을 제한함.
(기관에 등록된 앱에서 API별로 호출 가능한 총 건수를 제한함)
4. 시간제한 : 시작시간 ~ 종료시간 까지만 호출이 가능하게 제어함.
5. 요일제한 : 시작요일 ~ 종료 요일까지만 호출이 가능하게 제어함.
