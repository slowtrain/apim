크라우드펀드 자금관리

(Crowd Fund Management)

크라우드펀드 자금관리 API 를 사용하면 투자자의 청약증거금을 편리하고
안전하게 보관 가능한 고객 중심의 디지털 플랫폼을 구축할 수 있습니다.

주요 특징

고객에게 편리하고 안전한 투자경험을 제공할 수 있습니다.

신한금융투자 애널리스트가 작성한 시장관련 다양한 이슈를 다룬 리포트를 제공합니다.

  • 이용대상

    •   부동산(주택,아파트 ) 중개 플랫폼에서 금융서비스와 시너지를 높이고 싶은 기관

  • 서비스 특징

    •   이용기관의 온라인 채널에서 상품(매물)탐색과 함께 금융한도 조회 및 신청가능
    •   금융신청 정보 등 기반으로 중개매물에 대한 계약상태 관리 가능
      (상품플랫폼을 통해 중개매물 확인 후, 계약은 양도인과 직거래로 진행되는 리스트 관리 가능)

  • 수수료

    •   수수료 납부 방식 : 매월 10일 제휴기관 수수료 결제 계좌에서 출금 (휴일인 경우 익영업일 출금)
    •   수수료 부과 대상 : 대출 승인 내역 조회 API (건당 100원)
